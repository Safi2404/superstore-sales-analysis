# Superstore Sales Analysis

## Overview

This project analyzes retail sales data to extract business insights across categories, regions, and customer segments using Python.

## Objectives

* Understand sales distribution across categories
* Identify top-performing products
* Analyze regional performance
* Segment customers based on contribution to sales

## Tools & Technologies

* Python
* Pandas
* Matplotlib

## Dataset

Superstore sales dataset containing:

* Orders, customers, regions
* Product categories and sub-categories
* Sales values

## Key Steps

1. Data Cleaning

   * Removed inconsistencies
   * Standardized column names

2. Data Analysis

   * Total sales calculation
   * Category-wise and region-wise aggregation
   * Customer segment analysis

3. Feature Engineering

   * Created a derived `Profit` column using assumed margin

4. Visualization

   * Bar charts for category and regional performance

## Key Insights

* Certain categories contribute disproportionately to total sales
* Regional variation exists in sales performance
* A small subset of products drives significant revenue

## How to Run

1. Open the notebook `sales_analysis.ipynb`
2. Run all cells
